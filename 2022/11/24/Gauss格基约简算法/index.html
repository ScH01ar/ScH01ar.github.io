Gauss格基约简算法

由Gauss提出的二维格基约化算法

算法伪代码

即对给定的两个基向量进行不断的相互约化，最终求得格上的最小向量 Loop (a) If ||v2|| < ||v1||, swap v1, v2 (b) Compute m = ⌊ v1∙v2 / v1∙v1 ⌉ (c) If m = 0, return v1, v2 (d) v2 = v2 - m*v1 Continue Loop

思路

给定公钥q、h，以及明文e 要求私钥f、g 他们之间有如下关系： fh ≡ g mod q 即 kq + g =fh g = fh -kq 根据这一特点即可构造格，也就是格密码中很经典的，遇到一个向量乘矩阵，即可进行构造 由于存在一组a、b，使得a（1，h）+ b（0，q）=（f，g） a = f，b = k 选取一组基底为（1，h）和（0，q），也就是上述高斯约简算法当中的v1与v2，求出v1与v2格上最小的向量，即为（f，g）
